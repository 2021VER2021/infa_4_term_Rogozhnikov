Форматы обмена данными, такие как JSON(JavaScript Object Notation) нужны для того, 
чтобы облегчить работу с данными. Конкретно, использование единого формата обмена 
данных позволяет:
1) Легко пользоваться и передавать данные между программами, написанными на разных
   языках
2) Легко парсить данные
3) JSON распространён в web-разработке, поэтому с его помощью можно получать
 данные из интернета (данные в интернете представлены в данном формате, по крайней
 мере, курс биткоина)

 Таким образом форматы обмена данными существуют для единообразного представления 
 данных, и единообразной работы с ними.
